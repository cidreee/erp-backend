# Mini-ERP Backend 

Sistema de backend hecho en Java Spring Boot para gestionar inventario, ventas, facturación e instalaciones.

## Tecnologías
- Java 21
- Spring Boot 3.5
- MySQL
- JPA / Hibernate
- REST API

## Endpoints iniciales
- `GET /products`
- `POST /products`
- (más adelante: `/sales`, `/inventory`, `/installations`...)
